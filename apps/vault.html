To login and access the Vault, create a personal access token in your GitHub user settings, selecting only scopes:

  • read:org

  • read:user

WARNING: You must keep your access token secure.

Open the page https://vault.lsst-sandbox.ncsa.illinois.edu/, select the GitHub sign in method, and enter your token.
